# APIドキュメント

### タイムラインの取得

Endpoint

```
GET /api/v0/tl
```

Request

```json
{}
```

Response

```json
[
  {
    "id": 1,
    "title": "foo_bar_baz",
    "prefecture": {
      "id": 1,
      "name": "北海道"
    },
    "budget": {
      "id": 1,
      "range": "0 - 1,500円"
    },
    "target": {
      "id": 1,
      "name": "どちらでも"
    }
  },
  ...
]
```
